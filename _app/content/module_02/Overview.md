# Module 2: How to Use MySQL Workbench

## Module Overview
This module introduces MySQL Workbench, a powerful tool for designing, developing, and administering MySQL databases. You will learn how to install, configure, and use Workbench for real-world database tasks.

### Learning Outcomes
- Install and configure MySQL Workbench
- Connect to a MySQL server
- Use Workbench to create and manage databases and tables
- Run SQL scripts and queries
