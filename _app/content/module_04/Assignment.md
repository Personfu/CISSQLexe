**Learning Outcomes:**

## Assignment Instructions

### Exercise 1: INNER JOIN
Write a SQL statement to list all orders with customer names using an INNER JOIN between `Orders` and `Customers`.

### Exercise 2: LEFT JOIN
Write a SQL statement to list all customers and their orders, including customers with no orders.

### Exercise 3: Multi-table JOIN
Write a SQL statement to list order details, including customer name and product name, using JOINs across `Orders`, `Order_Items`, `Products`, and `Customers`.
