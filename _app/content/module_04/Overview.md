# Module 4: How to Retrieve Data from Two or More Tables

## Module Overview
This module covers SQL techniques for retrieving data from multiple tables using JOINs. You will learn to write INNER JOIN, LEFT JOIN, and other multi-table queries, and apply them to real-world scenarios.

### Learning Outcomes
- Write SQL queries using INNER JOIN, LEFT JOIN, RIGHT JOIN
- Retrieve and combine data from multiple tables
- Understand join conditions and their impact on results
