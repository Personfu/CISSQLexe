# Module 1: Introduction to Relational Databases

## Module Overview
Welcome to Module 1! In this module, you will learn the foundational concepts of relational databases, their structure, and why SQL is essential in the real world.

### Learning Outcomes
- Define what a relational database is
- Describe the purpose and structure of tables, rows, and columns
- Explain the importance of primary keys and relationships
- Understand the basics of SQL
