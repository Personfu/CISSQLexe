# Module 3: Retrieving Data from a Single Table

## SELECT Basics
- Retrieve all columns: `SELECT * FROM table_name;`
- Retrieve specific columns: `SELECT column1, column2 FROM table_name;`

## Filtering Results
- Use WHERE to filter: `SELECT * FROM table_name WHERE condition;`
- Comparison operators: =, <, >, <=, >=, <>, BETWEEN, IN, LIKE

## Sorting and Limiting
- Sort results: `ORDER BY column ASC|DESC`
- Limit results: `LIMIT n`

## Aliasing Columns
- Rename columns in results: `SELECT column AS alias FROM table_name;`
