# Module 3 Assignment: Retrieve Data from a Single Table

**Course:** SQL Essentials for the Real World
**Module:** 3 – Retrieve Data from a Single Table
**Learning Outcomes:**
- Write SELECT statements to retrieve and filter data (CLO 1,2)

---

## Assignment Instructions

### Exercise 1: Basic SELECT
Write a SQL statement to select all columns from the `Products` table.

### Exercise 2: Filtering
Write a SQL statement to select all products with a price greater than $100.

### Exercise 3: Sorting
Write a SQL statement to select product names and prices, sorted by price descending.

### Exercise 4: Aliasing
Write a SQL statement to select product names as `Product`, and prices as `Cost`.
