# Module 3: How to Retrieve Data from a Single Table

## Module Overview
This module teaches you how to use SQL SELECT statements to retrieve data from a single table. You will learn filtering, sorting, and formatting results for real-world use.

### Learning Outcomes
- Write SELECT statements to retrieve data
- Use WHERE, ORDER BY, and LIMIT clauses
- Format and alias result columns
