**Course:** SQL Essentials for the Real World
**Module:** 8 – How to Back Up and Restore a Database
**Learning Outcomes:**
- Perform and test backup and restore operations (CLO 1,2,3)

---

## Assignment Instructions

### Exercise 1: Full Backup
Write the command to perform a full backup of a database named `mydb` using mysqldump.

### Exercise 2: Restore
Write the command to restore the backup to a new database named `mydb_restored`.

### Exercise 3: Recovery Plan
Describe a recovery plan for a production MySQL database, including backup frequency and testing.
