# Module 10 Assignment: How to Code Subqueries

**Course:** SQL Essentials for the Real World
**Module:** 10 – How to Code Subqueries
**Learning Outcomes:**
- Write subqueries for filtering and computation
- Use `ALL`, `ANY`, `IN`, and `EXISTS`
- Build correlated subqueries for row-aware logic

---

## Instructions
Complete the following tasks.

### Exercise 1: Scalar subquery
Write a query that returns invoice numbers for invoices whose total is greater than the average invoice total.

### Exercise 2: `IN` and `ANY`
Write a query that finds vendors with an invoice total greater than any invoice for vendor 95.

### Exercise 3: Correlated subquery
Write a query that returns vendor names that have invoices with payment totals less than the average payment total for their vendor.
