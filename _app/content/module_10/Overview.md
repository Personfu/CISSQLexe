# Module 10: How to Code Subqueries

## Module Overview
This module introduces subqueries for filtering, comparisons, and correlated row-level logic. You will learn how subqueries enable complex queries without additional tables.

### Learning Outcomes
- Understand scalar and table subqueries
- Use `IN`, `ANY`, `ALL`, and `EXISTS`
- Implement correlated subqueries for row-aware filtering

---

## What You'll Learn
- When to use subqueries versus joins
- How to write nested queries that reference another query result
- Practical patterns for business reporting and condition evaluation
