# Module 5 Examples

## Example 1: Insert a Product
```sql
INSERT INTO Products (product_id, product_name, price) VALUES (2, 'Bass Guitar', 299.99);
```

## Example 2: Update a Product
```sql
UPDATE Products SET price = 349.99 WHERE product_id = 2;
```

## Example 3: Delete a Product
```sql
DELETE FROM Products WHERE product_id = 2;
```
