## Assignment Instructions

### Exercise 1: Insert Data
Write SQL statements to insert three new products into the `Products` table.

### Exercise 2: Update Data
Write SQL statements to update the price of one product and the name of another.

### Exercise 3: Delete Data
Write SQL statements to delete a product by its ID.

### Exercise 4: Data Integrity
Explain how you would ensure data integrity when performing these operations.
