# Module 5: How to Insert, Update, and Delete Data

## Module Overview
This module covers SQL statements for inserting, updating, and deleting data in tables. You will learn to use INSERT, UPDATE, and DELETE commands, and apply them to real-world scenarios using provided scripts and solutions.

### Learning Outcomes
- Write SQL statements to insert, update, and delete data
- Use WHERE clauses to target specific rows
- Understand the impact of DML operations on data integrity
