## Instructions
Complete the following tasks using MySQL data type design.

### Exercise 1: Table design
Create a `users` table with columns for email, first name, last name, date of birth, and signup date. Use appropriate MySQL data types and constraints.

### Exercise 2: Numeric design
Create a `transactions` table with a numeric amount column that stores values up to 9999.99 and a status code field.

### Exercise 3: Apply constraints
Modify the `users` table so `first_name` is `NOT NULL` and has a maximum length of 20 characters.
