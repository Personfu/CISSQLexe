# Module 9 Assignment: How to Code Summary Queries

**Course:** SQL Essentials for the Real World
**Module:** 9 – How to Code Summary Queries
**Learning Outcomes:**
- Apply aggregate functions to summarize data
- Write GROUP BY and HAVING queries
- Use window functions for advanced analysis

---

## Instructions
Complete the following tasks using the sample invoicing dataset.

### Exercise 1: Aggregate Sales
Write a query that returns each vendor, the number of invoices, and the total invoice amount. Filter to vendors with total invoice amount greater than 15,000.

### Exercise 2: Grouped Filtering
Create a query that finds invoice dates with more than two invoices and average invoice totals above 8,000.

### Exercise 3: Window Analysis
Write a query that returns each invoice row along with the running total of invoice amounts per vendor, ordered by invoice date.
